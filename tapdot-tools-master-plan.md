### UTMBuilder
#### tools.tapdot.org/marketing/utm

**Meta**
```
title:       UTMBuilder — UTM Parameter Generator | tapdot
description: Build and manage UTM tracking links. History saved locally.
             No data sent to any server. Free alternative to Google's Campaign URL Builder.
```

**What it does:** Build UTM-tagged URLs for campaign tracking. Source, medium, campaign,
term, content fields. Instant preview of the final URL. History of last 20 links saved
to localStorage. Bulk mode: paste a list of base URLs and apply the same UTM parameters

**Implementation**
```javascript
function buildUTM({ url, source, medium, campaign, term, content }) {
  const base   = new URL(url);
  const params = { utm_source: source, utm_medium: medium,
                   utm_campaign: campaign, utm_term: term, utm_content: content };
  Object.entries(params).forEach(([k, v]) => {
    if (v) base.searchParams.set(k, v.toLowerCase().replace(/\s+/g, '_'));
  });
  return base.toString();
}
```

**Gap:** Google's own Campaign URL Builder is the most-used tool but runs on Google's
servers and logs your campaign structure. UTMBuilder is local — your campaign
architecture stays private.

### HeadlineScore
#### tools.tapdot.org/marketing/headline

**Meta**
```
title:       HeadlineScore — Headline Analyser | tapdot
description: Score your headline for clarity, emotional pull, and SEO potential.
             Get 3 improved alternatives. On-device AI. Your copy stays private.
```

**What it does:** Paste a headline or blog title. Get a score (0–100) across four
dimensions — clarity, emotional pull, specificity, SEO keyword strength — plus
3 rewritten alternatives generated by Chrome's Prompt API on-device.

**Scoring logic (rule-based, no AI needed for the score itself)**
```javascript
function scoreHeadline(text) {
  const words   = text.trim().split(/\s+/);
  const len     = words.length;

  // Clarity: 6-12 words is optimal for headlines
  const clarity = len >= 6 && len <= 12 ? 100
                : len < 6 ? len * 14
                : Math.max(0, 100 - (len - 12) * 8);

  // Emotional pull: check for power words
  const POWER_WORDS = ['proven','secret','instantly','free','guaranteed','best',
    'ultimate','easy','fast','boost','transform','discover','unlock','surprising',
    'essential','avoid','warning','mistake','never','always','you','your'];
  const emotional = Math.min(100, POWER_WORDS.filter(w =>
    text.toLowerCase().includes(w)).length * 20);

  // Specificity: numbers and concrete details boost score
  const hasNumber  = /\d/.test(text);
  const hasColon   = text.includes(':');
  const hasQuotes  = /["']/.test(text);
  const specific   = (hasNumber ? 40 : 0) + (hasColon ? 30 : 0) + (hasQuotes ? 30 : 0);

  // SEO: moderate length, no stop-word stuffing
  const seo = len >= 7 && len <= 11 ? 80 : len >= 5 && len <= 13 ? 60 : 40;

  const total = Math.round((clarity + emotional + Math.min(100,specific) + seo) / 4);
  return { total, clarity: Math.round(clarity), emotional, specific: Math.min(100,specific), seo };
}
```

**AI alternatives (Chrome Prompt API)**
```javascript
const HEADLINE_PROMPT = (headline, scores) => `You are a marketing copywriter.
Rewrite this headline in 3 different ways that score higher on clarity,
emotional pull, and specificity. Keep each under 12 words.
Original: "${headline}"
Current weaknesses: clarity ${scores.clarity}/100, emotional ${scores.emotional}/100.
Output ONLY 3 alternatives, one per line, no numbering, no quotes.`;
```

**Gap:** CoSchedule Headline Analyzer sends your headline to their servers and tracks
what you write. HeadlineScore scores locally and rewrites with on-device AI.

### CompoundCalc
#### tools.tapdot.org/finance/compound

**Meta**
```
title:       CompoundCalc — Compound Interest Calculator | tapdot
description: Calculate compound interest with monthly contributions. Amortisation
             chart. Your financial data never leaves your browser.
```

**What it does:** Principal, annual rate, compounding frequency, time period, monthly
contribution → final value, total interest earned, total contributions. Year-by-year
breakdown table. SVG line chart showing growth over time. Compare two scenarios side
by side (e.g. different rates or contribution amounts).

**Implementation**
```javascript
function compoundInterest({ principal, rate, frequency, years, monthlyContrib }) {
  const n        = frequency; // compounds per year
  const r        = rate / 100;
  const t        = years;
  const pmt      = monthlyContrib;
  const results  = [];

  let balance = principal;
  for (let year = 1; year <= t; year++) {
    // Compound principal
    balance = balance * Math.pow(1 + r/n, n);
    // Add monthly contributions compounded through the year
    balance += pmt * 12 * ((Math.pow(1 + r/n, n) - 1) / (r/n));
    results.push({
      year,
      balance:      Math.round(balance),
      contributions: Math.round(principal + pmt * 12 * year),
      interest:      Math.round(balance - principal - pmt * 12 * year),
    });
  }
  return results;
}
```

### MortgageCalc
#### tools.tapdot.org/finance/mortgage

**Meta**
```
title:       MortgageCalc — Mortgage & EMI Calculator | tapdot
description: Calculate mortgage payments, total interest, and full amortisation
             schedule. Compare loans. Your home purchase details stay private.
```

**What it does:** Loan amount, interest rate, term, down payment → monthly EMI,
total payment, total interest, interest-to-principal ratio. Full amortisation
table (month by month). Overpayment calculator: "if I pay £200/month extra, how
much do I save?". Compare two mortgage offers side by side.

**Implementation**
```javascript
function calcEMI({ principal, annualRate, years }) {
  const r  = annualRate / 100 / 12;
  const n  = years * 12;
  if (r === 0) return principal / n;
  return principal * r * Math.pow(1+r, n) / (Math.pow(1+r, n) - 1);
}

function amortisationSchedule({ principal, annualRate, years }) {
  const emi    = calcEMI({ principal, annualRate, years });
  const r      = annualRate / 100 / 12;
  const months = [];
  let   balance = principal;

  for (let m = 1; m <= years * 12; m++) {
    const interest   = balance * r;
    const principalP = emi - interest;
    balance         -= principalP;
    months.push({
      month:     m,
      emi:       Math.round(emi),
      interest:  Math.round(interest),
      principal: Math.round(principalP),
      balance:   Math.max(0, Math.round(balance)),
    });
  }
  return months;
}
```

### CurrencyConvert
#### tools.tapdot.org/finance/currency

**Meta**
```
title:       CurrencyConvert — Currency Converter | tapdot
description: Convert between 170+ currencies. Rates fetched once and cached locally.
             No tracking of what you convert or how much.
```

**Note on architecture:** This is the one tool in the collection that requires a
single network request — for exchange rates. Use the Open Exchange Rates free tier
(1000 requests/month) or exchangerate-api.com free tier. Rates are fetched once
per day and cached in localStorage. The conversion itself is then entirely local.

**Implementation**
```javascript
const RATES_KEY    = 'tapdot-currency-rates';
const RATES_DATE   = 'tapdot-currency-date';
const API_URL      = 'https://open.er-api.com/v6/latest/USD'; // free, no key needed

async function getRates() {
  const today  = new Date().toDateString();
  const cached = localStorage.getItem(RATES_KEY);
  const date   = localStorage.getItem(RATES_DATE);

  if (cached && date === today) return JSON.parse(cached);

  const resp  = await fetch(API_URL);
  const data  = await resp.json();
  localStorage.setItem(RATES_KEY, JSON.stringify(data.rates));
  localStorage.setItem(RATES_DATE, today);
  return data.rates;
}

function convert(amount, from, to, rates) {
  // Convert via USD as base
  const inUSD = amount / rates[from];
  return (inUSD * rates[to]).toFixed(4);
}
```

**Privacy note in UI:** "Exchange rates are fetched once daily from open.er-api.com.
Only the rate request is made — the amounts you convert are never sent anywhere."

```javascript
    const prev = new Date(cycles[i].start);
    const curr = new Date(c.start);
    return Math.round((curr - prev) / 86400000);
```

```javascript
function generatePalette(hexBase) {
  const hsl  = hexToHSL(hexBase);
  return {
    primary:    hslToHex({ ...hsl }),
    light:      hslToHex({ ...hsl, l: Math.min(95, hsl.l + 40) }),
    dark:       hslToHex({ ...hsl, l: Math.max(10, hsl.l - 30) }),
    accent:     hslToHex({ ...hsl, h: (hsl.h + 30) % 360 }),
    complement: hslToHex({ ...hsl, h: (hsl.h + 180) % 360 }),
  };

function hexToHSL(hex) {
  const { r, g, b } = hexToRGB(hex);
  const rN = r/255, gN = g/255, bN = b/255;
  const max = Math.max(rN, gN, bN), min = Math.min(rN, gN, bN);
  let h, s, l = (max + min) / 2;
  if (max === min) { h = s = 0; }
  else {
    const d = max - min;
    s = l > 0.5 ? d / (2 - max - min) : d / (max + min);
    switch (max) {
      case rN: h = ((gN - bN) / d + (gN < bN ? 6 : 0)) / 6; break;
      case gN: h = ((bN - rN) / d + 2) / 6; break;
      case bN: h = ((rN - gN) / d + 4) / 6; break;
    }
  }
  return { h: Math.round(h*360), s: Math.round(s*100), l: Math.round(l*100) };
}
```

## Privacy policy additions for new collections

Add these sections to `tools.tapdot.org/privacy.html`:

```
Health tools note: Health data entered into BMICalc, MedicationLog, SymptomDiary,
CycleTracker, WaterIntake, and SleepLog is stored only in your browser's localStorage.
This data is classified as sensitive personal data under GDPR, HIPAA, and India's DPDP
Act. It is never transmitted to any server, never shared with any third party, and
never used for any purpose beyond displaying it back to you.

Finance tools note: Financial data entered into any tapdot finance tool — including
income, salary, investment holdings, and loan details — is processed entirely in your
browser and never transmitted anywhere. Currency exchange rates are fetched once daily
from a public API (open.er-api.com) — only the rate request is made, not your amounts.

Legal tools note: Text pasted into ContractRead is processed by on-device AI
(Chrome's built-in Gemini Nano). It is never sent to any server. Legal templates
(NDA, Privacy Policy, Terms) are generated entirely from local code.
```
